Crawler (HH.ru Job Scraper)
Этот проект представляет собой автоматизированный парсер вакансий с сайта hh.ru. Скрипт собирает данные о вакансиях, анализирует статистику по компаниям и выгружает результаты в структурированный Excel-файл.

Технологический стек
Node.js — среда выполнения.

Playwright — инструмент для автоматизации браузера и обхода защиты сайта.

ExcelJS — библиотека для создания и форматирования файлов .xlsx.

Инструкция по запуску
1. Подготовка
   Убедитесь, что у вас установлен Node.js.

2. Установка зависимостей
   Откройте терминал в папке проекта и выполните команду для установки всех необходимых библиотек:

Bash


npm install
Примечание: При первом запуске Playwright автоматически скачает необходимые бинарные файлы браузера.

3. Запуск парсера
   Для запуска процесса сбора данных используйте команду:

Bash


npm start
4. Результат
   После завершения работы скрипта в корневой папке проекта появится файл Top_20_Companies.xlsx. Он содержит два листа:

Топ-20 компаний — список работодателей с наибольшим количеством вакансий.

Вакансии топ-20 — детализированный список всех вакансий, найденных для этих компаний.

Возможные проблемы
Блокировки: Сайт hh.ru использует продвинутые системы защиты от ботов. Если скрипт возвращает пустые данные или не может загрузить страницу, попробуйте запустить его повторно через некоторое время (смена IP-адреса может помочь).

Видимость браузера: Скрипт работает в режиме headless: false, чтобы вы могли визуально контролировать процесс сбора данных и видеть, как скрипт закрывает всплывающие окна (регион, cookies).

Особенности реализации
В ходе разработки была реализована логика обработки динамических модальных окон, которые блокируют доступ к контенту на старте сессии. Также внедрен алгоритм агрегации данных из поисковой выдачи, что позволяет получать актуальную статистику без обращения к нестабильным каталогам компаний.
